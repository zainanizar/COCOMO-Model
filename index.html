What is COCOMO Models


The proposed product is an Enhanced Model for Software Effort Estimation. Software estimation is the mechanism of predicting the cost, effort, and duration required to develop the software. Software cost estimation plays a vital role in software engineering, as its success or failure depends entirely on it. Cost estimation's deliverables, like staff requirements, schedule, and effort, are essential information for the project's formation and execution. The model uses a regression formula to estimation cost using historical data with present and future characteristics.

Types of models

Basic Model

Used for early rough estimates of project cost, productivity, performance and schedule

Intermediate Model

Takes basic COCOMO as a starting point.
Uses effort adjustment factor which is calculated as the product of 15 multipliers (cost drivers).

Enhanced Function Point Model

Uses different effort multipliers for each phase of a project.
